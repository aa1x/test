⌈天才们的邀约⌋ 辅助答题工具

脚本功能：用绿色描边标出答案方便选择

如何使用脚本
方法一：使用 篡改猴(Tampermonkey) 插件。 方法二：打开网站后在控制台粘贴指令（无需插件）。
方法一： 
https://gh-proxy.org/https://github.com/89156/userjs/raw/refs/heads/main/user.js
已复制
使用方法：复制上方链接 → 打开 Tampermonkey 管理面板 → 实用工具 → 从 URL 安装
方法二：控制台指令 
fetch("https://gh-proxy.org/https://raw.githubusercontent.com/89156/userjs/refs/heads/main/quiz_highlight.js").then(r=>r.text()).then(eval)
已复制
使用方法：进入小游戏页面 → 按 F12 或右键空白点击检查打开开发者工具 → 切换到 Console（控制台） → 粘贴以上指令并回车执行。
快速说明
  • 脚本效果：在小游戏页面将正确答案用绿色描边标记，便于点击选择。
  • 使用 Tampermonkey 时请确保脚本来源可信并检查源码。
